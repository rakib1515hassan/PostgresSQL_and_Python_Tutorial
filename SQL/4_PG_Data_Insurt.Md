# Table Data Insert

## Syntax: Single Row Insert 

```base 
INSERT INTO table_name (column1, column2, ...) 
VALUES (value1, value2, ...);
```

## Syntax: Multiple Rows Insert Syntax

```base 
INSERT INTO table_name (column1, column2, ...) 
VALUES 
    (value1a, value2a, ...),
    (value1b, value2b, ...),
    (value1c, value2c, ...);

```
